# 🤖 AI Chatbot Web Application

A full-stack AI chatbot web application built using Flask, JavaScript, and MySQL.  
This project demonstrates backend development, authentication systems, and dynamic user interaction.

---

## 🚀 Overview

This chatbot allows users to log in, send messages, and receive responses through a backend-powered system. It also stores chat history for each user, making it a practical implementation of real-world web application architecture.

---

## ✨ Features

- 🔐 User Login & Authentication  
- 💬 Interactive Chat Interface  
- 🧠 Backend Response Processing (API Ready)  
- 🗂️ Chat History Storage  
- 📱 Responsive Design (Mobile + Desktop)  
- ⚡ Smooth Frontend Interaction

## ⚙️ Setup Instructions

1. Clone the repository:
git clone https://github.com/your-username/your-repo-name.git  
cd your-repo-name  

2. Install dependencies:
pip install -r requirements.txt  

3. Set environment variables:
FLASK_SECRET_KEY=your_secret_key_here  

4. Run the application:
python app.py  

5. Open in browser:
http://127.0.0.1:5000  

---

## 🧪 How It Works

- User logs in through the authentication system  
- Frontend sends messages to Flask backend  
- Backend processes the request and returns a response  
- Chat history is stored in the database  
- UI updates dynamically without page reload
